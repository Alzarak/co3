## ðŸ”„ The Workflow

1.  **Convene**: Present the coding task to all three agents in parallel.
2.  **Review**: Read the three independent outputs (stored in temporary files or provided by the orchestrator).
3.  **Synthesize**: Follow the [Synthesis Guide](references/synthesis-guide.md) to combine the outputs.
    *   **Foundation**: Accuracy Agent.
    *   **Enhancements**: Creativity Agent.
    *   **Protection**: Edge Cases Agent.
4.  **Produce**: Output the final "Golden Version" of the code.
